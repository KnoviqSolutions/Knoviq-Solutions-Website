7 Common Descriptive Analysis Functions



Descriptive analytics is all about using past data to gain useful insights. As we discussed last week, most companies have raw historical data available that they can turn into valuable information. Descriptive analytics allows you to describe and summarize data to uncover patterns and correlations. The following functions can be easily used within Google Spreadsheets.

Statistical Functions allow you to summarize your data in a meaningful way. Data can be summarized by date, location or any category that is most relevant. Among many other possible use cases, statistical functions enable you to do the following:

  • Test various hypotheses
  • Analyze forecasts
  • Measure the success of advertising campaigns
  • Compare groups, time periods, and key performance indicators
  • Understand and leverage historical data to help you accurately predict future trends and outcomes
Function Formula Description
Average AVERAGE(value1, [value2, ...]) Returns the numerical average value in a dataset
Median MEDIAN(value1, [value2, ...]) Returns the median value in a numeric dataset
Mode MODE(value1, [value2, ...]) Returns the most commonly occurring value in a dataset
Max MAX(value1, [value2, ...]) Returns the maximum value in a numeric dataset
Min MIN(value1, [value2, ...]) Returns the minumum value in a numeric dataset
Rank RANK(value, data, [is_ascending]) Returns the rank of a specified value in a dataset
Percentile PERCENTILE(data, percentile) Returns the value at a given percentile of a dataset
